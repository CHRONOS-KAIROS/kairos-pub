Welcome to MOIRAI The common interface for KAIROS
Multimodal Open Interfaces for Reasoning by Artiﬁcial Intelligence (MOIRAI) is the integration framework and access interface for DARPA’s KAIROS program. It collects services, data, and visualizations for each program performer type under a common access portal deployed inthe cloud atop scaled graph computing and data streaming technologies.

MOIRAI offers comprehensive commitment to KAIROS with a unified effort for not only TA3 System Integration and User Interface but also TA4 Data Creation for Development and Evaluation.



Major navigational elements are found by clicking on the hamburger menu in the top left corner of the screen. Here you can access program references, results from evaluations, and additional resources your team will need.
